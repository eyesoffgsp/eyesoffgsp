# How to view old versions of the website
## 1. Download the files
You will need to download the set of files from the point in time that you wish to see the website from. Here's how:
1. From this page, look on the right side for the Deployments section. Click **Deployments**
2. On the Deployments page, look for the version that you wish to see. Click the description of the change (e.g. "minor wording change") in blue
3. You will be taken to a preview of the specific code or image file that was changed. Click the **Browse Files** button on the top right of the page
4. You will be taken to what looks like the main repository page but has file versions from a previous point in time. Click the green **Code** button on the top right, and select **download ZIP**
  ## 2. View the webpage 
  Now you can view the old version of the webpage.
1. On your computer, navigate to the downloaded ZIP file and extract it
2. In the unzipped folder, locate the "index.html" file. Right click the file and open with browser (e.g. Firefox, Chrome, Edge, Safari, etc)
3. This version of the webpage will be opened in the browser
